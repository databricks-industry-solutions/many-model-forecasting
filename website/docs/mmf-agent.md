# MMF Agent

MMF is available as **skills** for AI coding assistants such as [Genie Code](https://docs.databricks.com/aws/en/genie-code/), [Claude Code](https://docs.anthropic.com/en/docs/claude-code/overview), and [Cursor](https://www.cursor.com/).

The MMF skills are a focused development kit that teaches AI assistants how to:

1. **Prepare and clean** time series data — data discovery, quality checks, and cleaning.
2. **Profile and classify** series — series profiling and classification (optional).
3. **Provision** the right Databricks clusters and forecasting resources.
4. **Execute** the forecasting pipeline.
5. **Post-process and evaluate** results.

All of this happens through natural language commands.

For installation instructions and full details, see the [skills README](https://github.com/databricks-industry-solutions/many-model-forecasting/tree/main/skills).

## Supported assistants

- Genie Code
- Claude Code
- Cursor
- GitHub Copilot
