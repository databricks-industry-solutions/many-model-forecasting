# Example Notebooks

The [examples](https://github.com/databricks-industry-solutions/many-model-forecasting/tree/main/examples) folder contains runnable notebooks for every supported frequency and model family.

## By frequency

- [Daily](https://github.com/databricks-industry-solutions/many-model-forecasting/tree/main/examples/daily)
- [Hourly](https://github.com/databricks-industry-solutions/many-model-forecasting/tree/main/examples/hourly)
- [Weekly](https://github.com/databricks-industry-solutions/many-model-forecasting/tree/main/examples/weekly)
- [Monthly](https://github.com/databricks-industry-solutions/many-model-forecasting/tree/main/examples/monthly)

## By scenario

- [External regressors](https://github.com/databricks-industry-solutions/many-model-forecasting/tree/main/examples/external_regressors) — forecasting with exogenous covariates.
- [Serverless](https://github.com/databricks-industry-solutions/many-model-forecasting/tree/main/examples/serverless) — running on serverless (including serverless GPU) compute.
- [M5](https://github.com/databricks-industry-solutions/many-model-forecasting/tree/main/examples/m5) — the M5 competition dataset.

## Post-evaluation analysis

After running `run_forecast`, use [post-evaluation-analysis.ipynb](https://github.com/databricks-industry-solutions/many-model-forecasting/blob/main/examples/post-evaluation-analysis.ipynb) for fine-grained model selection across your time series.
